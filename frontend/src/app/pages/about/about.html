Transforme seus sonhos de viagem em metas reais.

O FinGo nasceu da necessidade de precisão no planejamento de viagens. Unimos Inteligência Financeira com conectividade global para garantir previsibilidade, integrando taxas reais (CDI) e câmbio atualizado ao seu roteiro.

Pesquise

Encontre voos e hotéis reais através de nossas integrações com APIs globais.

Planeje

Defina sua meta financeira com projeções baseadas na taxa CDI vigente.

Decole

Acompanhe sua evolução em tempo real e transforme seu roteiro em realidade.

Arquitetura & Segurança

Autenticação JWT

O JSON Web Tokens garante sessões stateless seguras, fornecendo integridade em cada requisição.

Criptografia de Dados

Senhas e dados sensíveis são protegidos com hashing de alta performance, assegurando privacidade total.

Arquitetura MVC

Backend Node.js estruturado em Model-View-Controller com camada de Services para escalabilidade e performance.

Inteligência Financeira

Simulações baseadas em índices reais do mercado (CDI) para projetar a evolução do seu capital.

Orçamento Inteligente

O motor de busca integra dados em tempo real das APIs Booking.com e Kiwi, cruzando informações de hospedagem e voos para fornecer uma estimativa de custo realista. Diferente de calculadoras estáticas, o FinGo analisa tendências e preços vigentes para o seu destino.

Cálculo de Investimento

A inteligência financeira projeta metas utilizando a taxa CDI vigente, permitindo que o usuário visualize não apenas quanto precisa guardar, mas quanto seu dinheiro pode render até o embarque.

Estratégia CDI: Estimativa conservadora que ajuda a atingir a meta mais rápido do que na poupança.

Ecossistema de APIs

Serviço Provedor / API Finalidade Técnica
Países REST Countries Busca de dados demográficos e geográficos globais.
Câmbio API Awesome Cotações de moedas em tempo real para conversão financeira.
Taxas BrasilAPI (Taxas) Acesso aos índices CDI e Selic para projeções de investimento.
Persistência API Própria (Node/MongoDB) Armazenamento seguro de usuários, orçamentos e roteiros.
Roteiros API Geoapify Geoprocessamento e busca de pontos comerciais (Gastronomia/Compras).
Locais API Travel Places Curadoria de atrações turísticas, monumentos e locais históricos.
Hospedagem API Booking Integração de ofertas e disponibilidade de acomodações.
Voos Kiwi API Busca de passagens aéreas e conexões de voos globais.
